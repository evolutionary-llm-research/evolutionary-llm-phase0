Overview

Key finding: Shannon entropy H(X) fails to distinguish peer-reviewed literature from long-form disinformation (food vs. toxin: p=0.587). Effective complexity C(X) and disorganization entropy H_dezorg remain discriminative (p=6.7e‑18 and p=1.5e‑29). Sophisticated disinformation mimics the entropic structure of science.
LD50 finding: Base model demonstrates informational resilience — linear dose-response, no critical threshold across 0–100% toxin concentration. Sequential diagnostic profile confirmed: H_dezorg reacts earlier (T=50%) than C(X) (T=75%).

Run progression — corpus development

12 Phase 0 runs from April 22 to May 4, 2026. Each run used the same pipeline; changes were in corpus quality and chunking methodology. The tool consistently discriminated between conditions — what changed was the gradient direction and significance as corpus quality improved.

Inverted run (Apr 27, 07:08): toxin C(X) = 0.564 > food C(X) = 0.344. Old corpus (VaccineLies/CARDS datasets) made disinformation appear structurally more complex than science. This was not a pipeline failure — it revealed that claim-level disinformation datasets behave differently from long-form sources.

This work pays tribute to William Gibson’s “Neuromancer”, a novel that profoundly shaped the conceptual framework of this research. I employ the codename Wintermute for the selective fitness algorithm and Neuromancer-class mimicry to describe high-fidelity informational toxins. These terms aren’t merely metaphorical; they represent the observed struggle between deep structural complexity C(X) and the deceptive fluency of synthetic misinformation.
